Code buff is an application, which is used to have a single place to view all the coding contests happening across so many different platforms. This app aims to promote and help grow the competitive programming community worldwide. The app covers popular coding websites such as Codechef, Codeforces, Topcoder, HackerRank, HackerEarth and others. The app automatically updates all the coding contests beind held over these platforms and also allows users to set reminders Made with in india.
